hierarquia de titulos:

hierarquia de titulos pode ser comandada atrasvez das tags < h1 >, < h2 > até o maximo de h6

quando usar:

são usadas para categorizar os titulos de materias dando maior enfase na materia principal, utilizando a tag < h1 >

vale lembrar que as tags de hierarquia de titulos não podem ser utilizadas parar escrever textos, isso se deve ao tamanho do texto ser auterado atraves de CSS

semantica em HTMl

a semantica em HTML referece ao significado da palavra e não a funcionalidade, isso pq com o passar do tem as tecnologias vao mudando e com isso algumas tags utilizadas em html acabam caindo em desuso.. ficam obssoletas
por exemplo, comandos como < font > antes utilizada em html para configuração da fonte de escrita, acabou caindo em desuso por essa ser uma aplicação de CSS.
