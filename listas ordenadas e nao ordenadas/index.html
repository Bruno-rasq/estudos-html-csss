listas

listas ordenadas

comando < ol > + tag < li >

  1. hi
  2. hello
  3. hola
  4. oi

tabem é possivel definir qual o tipo de contado da lista, se é numero, letras ou simbolos ultizando comando TYPE=""

EXEMPLO: < ol type="A" >

  1. hi
  2. hello
  3. hola
  4. oi

tambem é possivel mudar o valor de qual a listagem começará, com o comando start=""

exemplo: < ol type="number" start="4" >

  1. hi
  2. hello
  3. hola
  4. oi

lista não ordenadas

utiliza a tag < ul > podendo utilizar complemento type=""

  • hi
  • hello
  • hola
  • oi
