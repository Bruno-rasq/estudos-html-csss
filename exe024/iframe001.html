iniciando estudos de iframe

iframes = inline-frame são frames, um bloco de imagem em linha. por padão um iframe possue um tamanho 300 x 150, podendo ser configurados.

utilizando iframe para acessar um site.


nem todo site permite uso de iframe, como youtube e outros. e tambem ixistem algumas incompatibilidades de dispositivos, alguns smarthphones, tables e etc

configuração de iframe:

dentro das configurações do iframe, caso sejam adicionados uma largura e altura, e lá no arquivo de CSS tbm sejam adicionados altura e largura, o que vai valer para o site será as atribuições feitas em css.
outra atribuição que vale citar é o "scolling" o scolling permite configurar se o iframe possuirá barra de rolagem ou não.

scolling > no = caso não queirá a barra.
scolling > yes = caso queirá
scolling > auto = configuração padrão, caso o site seja grande o suficiente ele terá a rolagem, caso contrario não
