listas

é possivel criar listas atraves de tags de paragrafos, mas este é uma forma meio burra de se fazer. um dos problemas é que se for necessario alterar algum item, tera de alterar todos os itens da lista.

para isso temos a tag < ol >

  1. io
  2. hello
  3. hola
  4. hi

podemos add um complemento a tag de lista ordenada identificando qual o tipo de lista, numeros, letras ou simbolos e em qual termo partira o item inicial.
para isso utilizamos < type="" > e < start="" >

exemplo: < ol type="number" start="2" >

  1. io
  2. hello
  3. hola
  4. hi

listas nao ordenadas

tag < ul > e pode ser complementada apenas com o comando type=""

exemplo; < ul >

  • io
  • hello
  • hola
  • hi

lista mista

é possivel criar listas mistas add uma tag < ol > dentro de uma tag de lista

exemplo:

  1. coisas que gosto:
    1. musicas
    2. programação
    3. desenhar
    4. musculação
  2. coisas que não gosto:
    1. calculo
    2. portugues
    3. multidoes

listas de definição

listas de definição são utilizadas quando se quer citar uma listasde itens, como por exemplo siglas, mantendo a sigla intacta e add o significado/definição. utiliza-se as tags < dt, dl, dd > sendo dt para titulo, dl para a lista e dd para a definição
