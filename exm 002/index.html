paragrafos

caso queira escrever um paragrafo posso utilizar das tags < p > e </p > assim tudo que for escrito dentro das tags fica em um só paragrafos

caso precise de uma quebra de texto posso utilizar a tag < br >

comentário dentro do vscode utiliza-se <! -- -- > dessa forma tudo que for escrito dentro da tag o vscode vai ler como comentário de codigo

simbolos;

® € ↓ ™ ¢ entre outros simbolos... utiliza-se o comando & seguido de um composto especifico para cada simbolo esmplo & copy; para gerar o simbolo ©
