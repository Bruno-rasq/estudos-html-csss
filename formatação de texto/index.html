Principais formatações

Negrito e Destacado

Nesta frase temos um termo em negrito utilizando a tag B (não semantica)

Nesta frase temos um termo em destaque utlizando a tag strong (semantica)


italico e enfase

Nesta frase temos um termo em italico utilizando a tag I(não semantica)

Nesta frase temos um termo em enfase utilizando a tag Em ( semantica)


recomendação

para HTML não faz muito sentido usar as tags para negrito e italico ja que são formatações de design de textos, neste caso é recomendado utilizar apenas as tags Strong e Em


marcado

podemos tambem usar a tag mark para marcar textos.

Texto excluido.

podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado.

Texto inserido

podemos marcar um texto insirido para dar enfase e indicar que ele foi adicionado depois


para excluir um texto usa-se a tag Del e para incluir um texto usa-se a tag ins

sublinhado

texto pode ser sublinhado usando a tag U (não semantico)

texto sobreescrito/subescrevito

para texto sobrescrito utiliza-se a tag sup


para texto subscrito utiliza-se a tag sub
