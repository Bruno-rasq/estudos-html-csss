listas mistas e de definição

listas mistas versão 1.0

funciona da mesma forma que as listas ordenadas e não ordenadas.

EXEMPLO: utilizando a tag < ol > dentro de uma tag < ol >

  1. numeros
    1. Aaaaa
    2. bbbbb
    3. cccvc
  2. letras
    1. •••••
    2. °°°°°
    3. ○○○○

codigo: 

        
<ol>
    <li>numeros</li>
        <ol type="number" start="1">
            <li>Aaaaa</li>
            <li>bbbbb</li>
            <li>ccccc</li>
            <li>ddddd</li>
        </ol>
    <li>letras</li>
        <ol type="A" >
            <li>Aaaaa</li>
            <li>bbbbb</li>
            <li>ccccc</li>
            <li>ddddd</li>
        </ol>
</ol>

listas de definição

utiliza-se das tags < dl, dt, dd > sendo dl a lista de definição em si, dt itens de lista que são titulos e dd item de definição.

HTMl
linguagem de marcação para criação de conteúdo para sites
CSS
linguagem de marcação para criação do design de sites

codigo:

        
<dl>
    <dt> HTMl </dt>
        <dd> linguagem de marcação para criação de conteúdo para sites</dd>
    <dt> CSS </dt>
        <dd> linguagem de marcação para criação do design de sites </dd>
</dl>
