links e âncoras em HTMl

links externos

são utilizados quando se quer deixar um link que redirecione o usuário para outra página.

utiliza-se a tag < a href="" >

Exemplo:

Youtube: leanbeffpatty

Codigo:

            
                <p> Youtube: <a href="https://youtu.be/NOGK-ovOp14"> leanbeffpatty chanel</a></p>

links com retorno

no primeiro caso, quando a pessoa clica no link ela é redirecionada para o site e este por si sobrepoe o site principal.

Para lidar com isso é aconselhado utilizar de complemento a tag os comandos target="_blak" rel="external" assim quando clicado no link, abre uma nova aba redirecionada e mantem a aba anterior com o site principal

Exemplo:

Youtube: Leanbeffpaty
